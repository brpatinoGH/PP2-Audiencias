# Sistema de Gestión de Audiencias - PP2

Este proyecto forma parte de mi **Práctica Profesionalizante 2** en la Tecnicatura Superior en Desarrollo de Software (IES). Es un sistema diseñado para la gestión y administración eficiente de audiencias, enfocándose en la robustez del backend y la escalabilidad de los servicios.

## 🚀 Tecnologías Utilizadas

* **Lenguaje:** Java 17+
* **Framework Principal:** Spring Boot
* **Gestión de Dependencias:** Maven
* **Herramientas de Desarrollo:** IntelliJ IDEA
* **Persistencia:** Spring Data JPA (PostgreSQL/MySQL) 
* **Arquitectura:** Microservicios

## 🛠️ Funcionalidades Principales

* **Gestión de Audiencias:** CRUD completo para el registro, edición y consulta de audiencias.
* **Lógica de Negocio:** Implementación de validaciones específicas para la asignación de horarios y salas.
* **API REST:** Endpoints estructurados para la integración con el frontend (Angular).

## 📁 Estructura del Proyecto

El repositorio se organiza principalmente en el módulo de backend:
- `src/main/java`: Contiene la lógica de controladores, servicios y repositorios.
- `src/main/resources`: Configuración de propiedades y base de datos.

## ⚙️ Configuración e Instalación

1.  **Clonar el repositorio:**
    ```bash
    git clone [https://github.com/brpatinoGH/PP2-Audiencias.git](https://github.com/brpatinoGH/PP2-Audiencias.git)
    ```
2.  **Configurar la base de datos:**
    Modificar el archivo `application.properties` con tus credenciales locales.
3.  **Ejecutar la aplicación:**
    Abrir el proyecto en IntelliJ IDEA y ejecutar la clase principal o usar:
    ```bash
    mvn spring-boot:run
    ```
